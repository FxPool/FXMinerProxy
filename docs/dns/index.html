前置条件：路由器管理权限

本方案通过修改矿场路由器的 DNS 解析记录，将矿池域名指向 FXMinerProxy 中转服务器。
运维人员需要拥有矿场路由器的管理员权限，才能配置自定义 DNS 规则。
整个过程不需要修改任何矿机配置，矿工端完全无感知。

部署流程

三步完成部署

整个过程最快 10 分钟即可完成，无需复杂的网络知识。

部署 FXMinerProxy 中转服务

在一台 Linux 服务器上一键安装 FXMinerProxy，配置好目标矿池地址和抽水钱包，设定抽水比例。

配置路由器 DNS 劫持规则

登录矿场路由器管理后台，将目标矿池的域名 DNS 解析指向 FXMinerProxy 服务器 IP，所有经过该路由器的矿机流量将自动被中转。

后台监控 稳定运行

通过 FXMinerProxy 可视化后台实时查看算力曲线、矿机在线数、抽水状态等数据，全程可控。
